===================================
Improving interoperability with pip
===================================

.. note::
   Conda 4.6.0 added improved support for interoperability between conda and pip. This involved the now deprecated ``prefix_data_interoperability`` configuration setting. For more information on this deprecated feature, see older versions of these `docs <https://docs.conda.io/projects/conda/en/26.1.x/user-guide/configuration/pip-interoperability.html>`_.

Conda now supports pip-installed packages using the ``conda-pypi`` package. For details on this package, see the `conda-pypi documentation <https://conda.github.io/conda-pypi/>`_.
