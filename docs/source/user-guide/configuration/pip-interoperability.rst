===================================
Improving interoperability with pip
===================================

.. note::
   Conda historically added improved support for interoperability between conda and pip with version 4.6.0. This involved the ``prefix_data_interoperability`` configuration setting. For more information on this deprecated feature, see an older version of this page `here <https://docs.conda.io/projects/conda/en/26.1.x/user-guide/configuration/pip-interoperability.html>`_.

Conda now supports pip-installed packages using the ``conda-pypi`` package. For details on this package, see the `conda-pypi documentation <https://conda.github.io/conda-pypi/>`_.
