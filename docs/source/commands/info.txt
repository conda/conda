Display information about current conda install.

**usage**: ``conda info [-h] [-a] [-e] [-l] [-s] [package_name [package_name ...]]``

*package_name*
        display information about packages

optional arguments:
  -h, --help     show this help message and exit
  -a, --all      show all information, (environments, license, and system information
  -e, --envs     list all known conda environments
  -l, --license  display information about local conda licenses list
  -s, --system   list environment variables
